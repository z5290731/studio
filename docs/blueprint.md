# **App Name**: MongoQuill

## Core Features:

- Access Code Entry: Landing page for entering a 6-digit access code.
- Database Selection: Display simulated MongoDB databases: Singapore Airlines and NewJeans.
- Query Editor: Code editor for entering MongoDB queries.
- Query Results Display: Display results of the MongoDB query.
- Simulated Datastore: Data is not actually persistent

## Style Guidelines:

- Primary color: Yellow (#ffdc00)
- Secondary color: White (#ffffff)
- Tertiary color: Black (#000000)
- Secondary Accent colors: Indigo (#3f61c4), Red (#ff635d), Green (#1ac987), Pink (#fa91b6)
- Tertiary Accent colors: Blue (#2daae2), Teal (#007882), Purple (#8a68c8), Orange (#ff8200)
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral look suitable for both headlines and body text.
- Code font: 'Source Code Pro' for displaying MongoDB queries.
- Use simple, line-based icons for database and action representations.
- Divide layout into clear panels for query input, database selection, and results display.
- Use simple transitions for database loading, feedback for when a user enters the access code correctly, and when queries are executed.
